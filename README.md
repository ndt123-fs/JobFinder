# Job Finder Application

A backend system for a web-based job platform built with Spring Boot,
enabling job posting, searching, CV upload, and automated email notifications.

## Tech Stack
- Backend: Java, Spring Boot, RESTful API, JWT, MySQL, Cron Job, Gradle

## Features
- JWT-based authentication with role-based access control (Admin/User)
- Job posting, searching with pagination and dynamic filtering
- File upload for CV submissions
- Scheduled Cron Jobs for automated job alert email notifications
